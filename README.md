實作過程：

依照畫面分析，橘色和綠色矩形，在橫向或直向的畫面上，兩者的間隙不變，與畫面左右兩側的間隙也不變，會隨畫面改變的是，矩形的寬度。

利用課堂上教的，藉由安插隱形固定寬度的矩形，產生間隙不變，寬度會變的效果。
